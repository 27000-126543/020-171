## 1. 产品概述

连锁口腔机构接诊话术质检看板——帮助运营经理发现各门店沟通薄弱环节的数据驱动平台。系统按门店、医生、项目类型展示接诊记录摘要，聚焦种植咨询、正畸初诊、儿童牙科、洁牙转化等场景下的常见问题分布，而非替医生说话。
- 解决核心问题：连锁口腔机构难以系统化发现和改善各门店接诊沟通质量
- 目标价值：降低因沟通不当导致的客诉率，提升高价项目转化率和患者满意度，将优秀话术从经验沉淀为可复制资产

## 2. 核心功能

### 2.1 用户角色
| 角色 | 进入方式 | 核心权限 |
|------|----------|----------|
| 运营经理 | 管理后台账号登录 | 查看全局数据、门店对比、管理话术库 |
| 区域负责人 | 区域账号登录 | 查看所辖门店数据、门店对比、标注质检 |
| 质检人员 | 质检账号登录 | 话术标签标注、撰写改进建议、提交优秀话术 |

### 2.2 功能模块
1. **接诊质检看板首页**：全局数据概览、场景分布、问题热力图、快捷入口
2. **话术标签标注页**：录音片段回放、标签选择、改进建议撰写、标注历史
3. **门店对比页**：多维度趋势对比、典型案例查看、培训需求识别
4. **优秀话术库页**：高满意度案例浏览、适用场景筛选、话术片段详情、培训引用

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|

| 优秀话术库页 | 培训引用按钮 | 一键生成培训材料引用格式，方便培训会直接使用 |

## 3. 核心流程

**质检标注流程**：质检人员登录 → 进入标注页 → 选择待标注录音 → 播放录音 → 选择话术标签 → 撰写改进建议 → 提交标注 → 标注计入门店数据

**门店对比流程**：区域负责人登录 → 进入门店对比页 → 选择对比门店 → 查看趋势图与场景维度对比 → 查看典型案例 → 识别培训需求 → 制定培训计划

**优秀话术沉淀流程**：质检人员/运营经理发现高满意度案例 → 提交至话术库 → 填写适用患者与使用边界 → 审核通过 → 进入话术库 → 被培训引用

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深青色(#0D9488)搭配暖橙(#F97316)作为强调色，传递专业感与活力
- 辅助色：浅灰蓝背景(#F0F4F8)、深墨色文字(#1E293B)、警示红(#EF4444)、成功绿(#10B981)
- 按钮风格：圆角8px、适度阴影、hover微动效
- 字体：标题使用 Noto Serif SC 衬线体传递专业信任感，正文使用 Noto Sans SC 保证阅读舒适度
- 布局风格：左侧固定导航 + 右侧内容区，卡片式模块布局，数据可视化区域留白充分
- 图标风格：线性图标，2px描边，统一使用Lucide图标库

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 数据概览卡片 | 4张统计卡片横排，渐变背景，大字号数字+小字号标签，入场渐现动画 |
| 首页 | 场景分布图 | 环形图 + 右侧图例，hover显示详情，配色对应场景类型 |
| 首页 | 门店健康度排名 | 横向条形图，渐变色条，hover显示分数明细 |
| 首页 | 问题标签热力图 | 热力矩阵，颜色深浅表示频次，tooltip显示具体数据 |
| 标注页 | 录音播放器 | 自定义播放控件，波形进度条，倍速切换按钮 |
| 标注页 | 标签选择面板 | 标签胶囊按钮，选中高亮，支持多选，分类展示 |
| 标注页 | 改进建议 | 文本域 + 快捷模板下拉，字符计数 |
| 对比页 | 趋势折线图 | 多条折线对比，面积填充，时间范围切换 |
| 对比页 | 典型案例卡片 | 左右分栏卡片，正面绿色边框/反面红色边框，场景标签 |
| 话术库页 | 话术卡片 | 双栏卡片，顶部场景标签，中间话术摘要，底部适用标签与评分 |
| 话术库页 | 详情弹窗 | 模态弹窗，分区展示话术文本/适用场景/使用边界/引用信息 |

### 4.3 响应式设计
- 桌面优先设计，主要适配1440px及以上分辨率
- 1280px以下：侧边导航收缩为图标模式，卡片改为单栏
- 768px以下：底部标签导航，全单栏布局，图表简化

### 4.4 3D场景指引
- 不适用，本项目为数据看板，无3D场景需求
